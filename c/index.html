Microphone/s

  1. Make sure Main Fader is at Unity
  2. Make sure Speaker Vol is at Unity
  3. Make sure XLR Mic is properly connected
  4. Unmute the channel the Mic is plugged into
  5. Make sure Mic Fader is at Unity
  6. Make sure L-R is pressed down
  7. If needed, turn up gain until the volume is adequate
  8. Microphone works
